## Für die AI-Konvertierung (Jocelyn)

- **PDF** = ein Bogen pro Datei (kompletter Aufriss, +3 mm Rand) → in Illustrator importieren.
- **Pink** = Schnitt-/Falzlinien (Hilfslinien, **nicht drucken**).
- Panel-Reihenfolge: Leimlasche · Seite Links · Front · Seite Rechts · Back + Flaps oben/unten.
- Text im PDF hat eingebettete Fonts (in AI bei Bedarf in Pfade/Kurven umwandeln).

## Noch offen / vor Druck prüfen

- **CE-Zeichen bewusst weggelassen** — erst aufdrucken, wenn gültiges Zertifikat + Testbericht vorliegen.
- **Filterlänge 4″** (`~305 mm`) ist unbestätigt — Werk-Rückfrage offen.
- Produktfoto-Platzhalter ersetzen, EAN je SKU einsetzen.
